// テーマ発想ブレスト
A場面イメージ
B障害イメージ
C動詞イメージ
D能力イメージ(物理表現、能力)

// UXからのシステム発想ブレスト
A~Dからランダムで1枚ずつ選び4種を組み合わせて下記に沿ってシステムを考える
・どのような新しい体験で
・どのような障害を
・どのように超えさせて
・どうユーザーを楽しませるか(達成感を与えるか)
※分かりやすくて、難しくない操作性

// システムの新パラメータ考案
UXからのシステム発想ブレストで考えたシステムは、
・どのような新パラメータを
・どのように処理して
・どのような判定を行うのか
その判定一連のシステムでどのような得点(達成感)としてユーザーにフィードバックされるのか

// プラットフォームを考え、入力方法と画像イメージの考案
・プラットフォーム
・画面のアクセプト比
・ゲームをどのように画面中で表現するのか
